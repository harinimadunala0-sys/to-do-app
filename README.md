# 📝 To-Do List GUI Application

##  Overview

This project is a simple and user-friendly To-Do List application built using Python and Tkinter. It helps users manage daily tasks efficiently through a graphical interface.

---

##  Features

* Add new tasks
* Delete existing tasks
* Mark tasks as completed
* Clear all tasks
* Persistent storage using file handling
* Scrollable task list for better usability

---

##  Technologies Used

* Python
* Tkinter (for GUI)
* File Handling

---

##  How It Works

* Tasks are stored in a Python list during runtime
* Data is saved into a text file (`tasks.txt`)
* When the application starts, it loads saved tasks
* GUI components like buttons and listbox handle user interaction

---

## How to Run

1. Make sure Python is installed
2. Download or clone the repository
3. Run the following command:

```
python todo.py
```

---

## Output

The application opens a graphical window where users can:

* Enter tasks
* View tasks in a list
* Perform actions using buttons
